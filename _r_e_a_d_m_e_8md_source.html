5 ## Introduction
6 
7 A simple library that improves the user experience when displaying images by Qt's QLabel and QGraphicsView. It also supports OpenCV Mat array type.
8 
9 smartGraphicsView supports show image functions which supports QImage and Mat (in BGR format). It also supports mouse clicks, moves, wheel events, and region selection.
10 
11 smartLabel supports show image functions which supports QImage and Mat (in BGR format). It also supports mouse clicks events.

13 ## Usage
14 
15 Functions below can be turned on or off individually.
16 
17 ### OpenCV
18 
19 If you wish to use OpenCV, add DEFINE += HAVE_OPENCV in your *.pro file.
20 
21 ### OpenGL
22 
23 If you wish to use OpenGL, you'll have to use OpenGL version (< QT 5.5) or add opengl module in the *.pro file (>= QT5.5). (QT += opengl)
24 
25 ### QSideButtonBar
26 
27 If you wish to turn off the SideButtonBar, add DEFINES += NO_SIDEMENU in your *.pro file.
28 
29 ### QtSVG
30 
31 If you wish to use icon for your QtSmartGraphicsView, add svg module in the *.pro file. (QT += svg)
